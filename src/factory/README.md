# Factory Method Pattern

## 概要

物流システムを作成する。

このシステムは配送方法によって、使用する輸送手段が異なる。

- トラック配送
- 船便配送

配送方法ごとに、生成するオブジェクトを切り替えたい。

この課題では、Factory Method パターンを使用して、
「オブジェクト生成処理の切り替え」
を学習する。

---

# お題

## 物流システムを作成する

配送システムには複数の配送方法が存在する。

### 陸路配送

- 輸送手段: トラック

### 海路配送

- 輸送手段: 船

配送処理は共通だが、
生成される輸送手段だけが異なる。

---

# 要件

## 輸送手段インターフェースを作成する

輸送手段は共通の操作を持つ。

例:

- 配送開始
- 配送完了
- 配送料計算
